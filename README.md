# CCNA Enterprise Network Simulation Lab

## Technologies

- VLANs
- DHCP
- ACLs
- NAT Overload (PAT)
- OSPF
- ISP Simulation

## Description

This Packet Tracer lab demonstrates VLAN segmentation, DHCP configuration, ACL implementation, NAT overload, OSPF dynamic routing, and Internet connectivity through a simulated ISP.

## Verification

- Successful OSPF route exchange
- Successful NAT translations
- End-to-End connectivity
- Internet access from internal hosts
